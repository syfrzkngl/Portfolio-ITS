Keahlian

UI-UX Design

Dua aspek penting dalam mendesain antarmuka pengguna dan pengalaman pengguna dalam aplikasi, situs web, atau perangkat lunak lainnya. Meski saling terkait, UI dan UX merujuk pada dua aspek berbeda dalam desain produk digital.

Web Development

Seseorang yang mengembangkan dan membangun situs web atau aplikasi web. Bertanggung jawab untuk menulis kode, merancang struktur, dan memastikan bahwa situs web atau aplikasi berfungsi dengan baik di berbagai perangkat dan platform. Pengembang web bekerja dengan berbagai bahasa pemrograman dan teknologi untuk mengimplementasikan desain dan fungsionalitas situs web.

MySQL DBMS

Perangkat lunak yang digunakan untuk mengelola dan mengatur data dalam basis data. DBMS memungkinkan pengguna untuk menyimpan, mengakses, memanipulasi, dan mengelola data secara efisien, aman, dan terstruktur. DBMS memainkan peran penting dalam mengelola data dalam aplikasi, situs web, dan sistem lainnya.
